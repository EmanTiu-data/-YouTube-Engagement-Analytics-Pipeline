# -YouTube-Engagement-Analytics-Pipeline
Developed a Python-based ETL pipeline using the YouTube Data API v3 to audit social media performance. The tool extracts real-time metadata and statistics from high-impact video content to calculate a custom 'Engagement Rate' metric. Using Pandas for data transformation and Matplotlib for professional visualization, the project identifies high-performing assets to guide data-driven marketing decisions.
